## 実行方法(macOS)
1. ターミナルを2つ立ち上げる
3. それぞれに以下の操作
- ターミナル1：  
  1. ファイルのあるディレクトリへ移動
  2. ターミナルに`irb`でRudyの世界へ
  3. ターミナルに`load "illust.rb"`でRubyのプログラムを読み込む
  4. ターミナルにtrueが帰って来ればおk
  5. defの実行ブロック名（`kansei`）を入力
  6. defのwriteimageの名前（kansei.ppm）で画像が出力される
- ターミナル2：
  1. ファイルのあるディレクトリへ移動
  2. できた画像を変換する（下のどちらか）
    -  →magick kansei.ppm kansei.png
    -  →convert kansei.ppm kansei.png
